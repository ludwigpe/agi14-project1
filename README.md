agi14-project1
==============

The greates game ever created by Megatron

Controls (Keyboard)
===================
UP:     Accelerate.

DOWN:   Brake.

LEFT:   Turn to the left.

RIGHT:  Turn to the right.

SPACE:  Jump.

F1:   Start current level.

F2:   Load Tron Level.

F3:   Load Ice Level.

F4:   Toggle Music.

ESC:  Exit Game.
